# Social Hub Assignment

This workspace is split into two folders:

- `frontend/` React + Vite social networking frontend
- `backend/` Node.js + Express admin panel backend

## Quick Start (from root)

Install dependencies:

```bash
npm run install:frontend
npm run install:backend
```

Run frontend:

```bash
npm run dev:frontend
```

Run backend:

```bash
npm run start:backend
```

Frontend URL:

- http://localhost:5173

Backend Admin URL:

- http://localhost:4000/admin

Default admin login:

- Username: `admin`
- Password: `Admin@123`

## Useful Root Scripts

- `npm run dev:frontend` start React dev server
- `npm run build:frontend` build frontend production bundle
- `npm run preview:frontend` preview frontend production build
- `npm run start:backend` start backend server
- `npm run dev:backend` start backend in watch mode
